**Policy as code identifier:** ALL_EXPOSED_SECRET_SUMOLOGIC

---

## This rule will fail

If a SumoLogic **access ID and/or access token** are present anywhere in the manifest:

```yaml
ACCESS_ID: sumoyz1701tigvjm5g8.-=>pi2kieiq6rvxjh
ACCESS_TOKEN: sumo18pbiz5||:z039evojioi226iveobhw3riof0hqkft13ip8mrl3zgnedf99doa3qwtysa9ijqm`
```

### Rule output in the CLI

```terminal
$ datree test *.yaml

>> File: failExample.yaml
❌ Prevent exposed secrets in SumoLogic objects [1 occurrence]
💡 Secret data found in manifest - keep your sensitive data elsewhere to prevent it from being stolen 
```
