## This rule will fail

If a JSON **web token** is present anywhere in the config:

```yaml
WEB_TOKEN: eyt7xf0ymd5rcy42n76d92wsigff80nmk.eym1vl26-qn6jrb4f1y9zg4sy5dfzka541gfr5/ll_250sg_/ku-ulv59iqkiao-.T6G4yz2ERwX5fd0BuiWAIHgvjVgBqa16B1WeCIxH0UQut1
```

### Rule output in the CLI

```terminal
$ datree test *.yaml

>> File: failExample.yaml
❌ Prevent exposed JWT secrets in objects [1 occurrence]
💡 Secret data found in config - keep your sensitive data elsewhere to prevent it from being stolen 
```
