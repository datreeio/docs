**Policy as code identifier:** ALL_EXPOSED_SECRET_AWS


---

## This rule will fail

If an AWS **access token** is present anywhere in the config:

```yaml
ACCESS_TOKEN: AGPAQPRC2KYLEMAQI3F6
```

### Rule output in the CLI

```terminal
$ datree test *.yaml

>> File: failExample.yaml
❌ Prevent exposed AWS secrets in objects [1 occurrence]
💡 Secret data found in config - keep your sensitive data elsewhere to prevent it from being stolen 
```
