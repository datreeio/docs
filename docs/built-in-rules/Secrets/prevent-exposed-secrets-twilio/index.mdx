**Policy as code identifier:** ALL_EXPOSED_SECRET_TWILIO

---

## This rule will fail

If a Twilio **API key** is present anywhere in the config:

```yaml
API_KEY: SKaB471B5FAe075E7D6b5F4db4822Da7bf
```

### Rule output in the CLI

```terminal
$ datree test *.yaml

>> File: failExample.yaml
❌ Prevent exposed Twilio secrets in objects [1 occurrence]
💡 Secret data found in config - keep your sensitive data elsewhere to prevent it from being stolen 
```
