**Policy as code identifier:** ALL_EXPOSED_SECRET_GITLAB


---

## This rule will fail

If a GitLab **Personal access token** is present anywhere in the config:

```yaml
ACCESS_TOKEN: glpat-37HjBcYY9Qb7RNnqm6wJ
```

### Rule output in the CLI

```terminal
$ datree test *.yaml

>> File: failExample.yaml
❌ Prevent exposed GitLab secrets in objects [1 occurrence]
💡 Secret data found in config - keep your sensitive data elsewhere to prevent it from being stolen 
```
