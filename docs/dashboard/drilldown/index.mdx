The [drilldown page](https://app.datree.io/drilldown) contains information about rules that failed when scanning your cluster.

### Filter table

The table on this page contains all the rules that failed in the last scan, along with information about the relevant resource. You can filter the table by namespace, resource kind or rule name.  

![drilldown-table](/img/dashboard/drilldown/table.png)

### Rule details

Click on a rule in the table to see more information about its impact, fix complexity and how to fix it.

![drilldown-rule-details](/img/dashboard/drilldown/rule-details.png)
